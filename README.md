# Rust Folder Menu

Кроссплатформенное приложение для выбора папки, написанное на Rust.

## Поддерживаемые платформы

- ✅ Windows (x64)
- ✅ Linux (x64) 
- ✅ macOS (Intel & Apple Silicon)

## Использование

Запустите программу и выберите папку в открывшемся диалоге.
```bash
# Сборка из исходников
cargo run
